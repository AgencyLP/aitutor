# AI Whiteboard Tutor (Demo)

A shareable, web-based demo of a PDF-grounded AI tutor:
- Full-screen whiteboard teaching (text + diagrams)
- Mini chat to interrupt and ask questions
- Strict "PDF-only" grounding with evidence drawer + PDF highlight
- Optional replay "video" mode (whiteboard recap)

## Requirements (for best experience)
- Desktop Chrome (WebGPU supported)
- A reasonably strong laptop (WebLLM runs on the user's device)

## Local development
```bash
npm install
npm run dev
